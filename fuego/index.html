Welcome to Kampler Linux


What is Kampler Linux?

Kampler Linux, shortly known as Kampler, is an upcoming Linux distribution based off the Debian GNU/Linux distribution, which uses the apt package management system. The project is currently run by koldfreg, and it is under development as of currently.
